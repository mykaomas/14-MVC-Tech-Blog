# Tech Blog CMS

## Description
This is a CMS-style blog site designed for developers who write about technology. It allows users to publish articles, blog posts, and share their thoughts and opinions on various tech-related topics.

## Technologies Used 
- Node.js
- Express.js
- MySQL
- Sequelize
- Handlebars.js
- Express Session
- Bcrypt
- Dotenv
